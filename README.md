# Weather app

## Description

This app displays the current weather for the current location. It uses the OpenWeather API.

## Technologies the app uses

<ul>
    <li>Swift</li>
    <li>SwiftUI</li>
    <li>CoreLocation</li>
    <li>Combine</li>
</ul>
